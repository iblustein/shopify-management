# 🔥 Stoked — AI Orchestration System
<!-- Maintained by the Stoked AI dev team -->

An autonomous multi-agent system that manages, develops, markets, and supports the Stoked Shopify store.

## Team Structure

### 🏢 Management
- **CEO Agent** — Command interface (WhatsApp), delegates tasks, reports back
- **COO Agent** — Cross-team coordination and task tracking
- **BI Agent** — Analytics and business intelligence across all teams

### 💻 Dev Team
- **PM Agent** — Translates business needs into dev tasks
- **Developer Agent** — Writes and commits code
- **QA Agent** — Reviews and tests before deploy
- **DevOps Agent** — Merges and deploys to Shopify

### 📣 Marketing Team
- **Analytics Agent** — Reads Shopify + ad platform data
- **Campaign Agent** — Manages Meta, Google, TikTok, Instagram
- **Email/SMS Agent** — Runs Klaviyo flows
- **Budget Guardian** — Enforces budget ceilings across channels

### 🎧 Customer Support Team
- **Triage Agent** — Categorizes incoming support requests
- **Support Agent** — Responds to customers
- **Escalation Agent** — Flags issues requiring human attention

## Control Interface
All commands are sent via WhatsApp to the CEO Agent.

## Tech Stack
- **Store:** Shopify + Printify
- **Code:** GitHub (this repo)
- **CI/CD:** GitHub Actions
- **Marketing:** Meta, Google Ads, TikTok, Instagram, Klaviyo
